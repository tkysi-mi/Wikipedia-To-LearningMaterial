## 技術的実装

### API エンドポイント

- `POST /api/auth/login` - 認証
- `POST /api/auth/logout` - ログアウト
- `GET /api/auth/me` - 認証状態確認
- `POST /api/materials` - 教材生成
- `POST /api/sessions` - セッション開始
- `POST /api/sessions/{id}/answers` - 回答送信
- `GET /api/sessions/{id}/result` - 結果取得

### 外部API連携

- **Wikipedia API**: 記事テキスト取得
- **OpenAI API (gpt-4o-mini)**: サマリ生成、○×問題生成

### データ管理

- **インメモリセッション管理**: セッション、回答データを一時保存（永続化なし）

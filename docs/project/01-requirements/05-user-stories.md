## メモ

### ユーザージャーニー全体（MVP実装完了版）

#### メインフロー（正常系）

1. **ベーシック認証** → パスワード入力 → 認証成功 ✅
2. **URL入力画面** → Wikipedia URLを入力 → 送信 ✅
3. **サマリ表示画面** → 3行サマリを読む → 「問題を開始」クリック ✅
4. **問題回答画面（1問目）** → ○/×を選択 → 正解/不正解フィードバック → 「次へ」クリック ✅
5. **問題回答画面（2-10問目）** → 同様に回答 ✅
6. **結果表示画面** → 正答数・正答率を確認 → 「別の記事で試す」クリック ✅
7. **URL入力画面に戻る** → 新しいURLで再学習 ✅

#### エラーフロー

- **無効なURL入力** → エラーメッセージ表示 → 再入力 ✅
- **Wikipedia API失敗** → エラーメッセージ表示 → 再試行 ✅
- **OpenAI API失敗** → エラーメッセージ表示 → 再試行 ✅

### 補足

- すべてのユーザーストーリーは**社会人（自習者）**を主要なペルソナとしています
- **学習履歴の永続化**や**ユーザー管理**は意図的に含めていません（デモ用途のため）
- MVP（US-001〜US-008）はすべて実装完了しました（2025-11-22）
- Phase 2のストーリーは今後のバックログリファインメントで詳細化します
